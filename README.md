# Biblioteca-POO

Operarea unei biblioteci.

Programul stocheaza atat abonatii biblliotecii, identificandu-i prin CNP, cat si cartile disponibile in biblioteca, identificate printr-un ID. Cartile au un stoc ce scade atunci cand un abonat imprumuta o carte si creste atunci cand abonatul o returneaza. In momentul returnarii se verifica daca termenul de restituire a fost depasit si, in caz afirmativ, se aplica o penalizare calculata in functie de numarul de zile intarziate. Abonatii nu pot imprumuta alta carte daca nu au abonamentul valabil sau au penalizari. Se calculeaza un scor de incredere pentru fiecare abonat bazat pe varsta, durata abonamentului si numarul de carti imprumutate. Daca scorul de incredere este trecut de 50/100, penalizarile au o reducere de 25%.
