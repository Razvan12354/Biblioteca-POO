# Biblioteca-POO

Clase pentru operarea unei biblioteci.

CERINTA:
    Realizati o aplicatie in c++ care sa permita gestionarea activitatii unei biblioteci. Includeti o alerta de reinnoire, un scor de incredere calculat pe baza activitatii clientului si o functie ce verifica atat data la care trebuie sa fie returnata o carte, cat si daca aceasta a trecut sau nu de data curenta.
