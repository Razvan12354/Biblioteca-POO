# Biblioteca-POO

Clase pentru operarea unei biblioteci.

CERINTA:
    Realizati o aplicatie in c++ care sa permita gestionarea activitatii unei biblioteci. Includeti un scor de incredere calculat pe baza activitatii clientului, un sistem de penalizari si un sistem de returnare ce aplica penalizari in cazul intarzierilor. Durata abonamentului si balanta de penalizari pot fi modificate.
